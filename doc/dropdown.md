 * 支持react native modal 除visible外的所有属性
 * 默认点击空白区域关闭模态框

##### props参数：

prop | type | default | required | description
---- | :----: | :-------: | :--------: | -----------
**closeTime** | `Number` | `500` | false | 设置收起动画时间
**extraTop** | `Number` | `0` | false | 距离顶部额外的高度设置
**openTime** | `Number` | `1000` | false | 设置展开动画时间
