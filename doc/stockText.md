用于显示收益率的文本,收益为正显示红色,收益为负显示绿色

##### props参数：

prop | type | default | required | description
---- | :----: | :-------: | :--------: | -----------
**containerStyle** | `*` |  | false | 容器样式
**fontSize** | `Number` | `14` | false | 字体大小
**value** | `String` |  | false | 当前数值,传入字符串类型
